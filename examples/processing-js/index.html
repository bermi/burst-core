JavaScript Ninja

a processing.js ninja controled by Burst-Core

The ninja character in this demo is being controlled by the Burst Engine core. Burst-Core brings time-line animation control to JavaScript. Based on the classic structure: Time-line > Track > Key-frame, Burst-Core is suitable for complex, multi-layered, non-linear animations. Burst-Core also supports method-chaining and call-back architecture to support rich, interactive use cases.

Source code: the following is the Burst-Core code from the Processing.js script

// BURST-CORE CHARACTER CONTROL
burst.timeline("ninja",1,40,0.05,true)
  .obj("processing_instance", p)
    .track("mouseX")
      .key(1,0)
      .key(5,400)
      .key(10,100)
      .key(15,300)
      .key(20,0)
      .key(25,400)
      .key(30,000)
      .key(35,400)
      .key(40,100)
    .track("mouseY")
      .key(1,0,"outBounce")
      .key(20,300)
      .key(30,140)
      .key(35,100,"inOutElastic")
      .key(40,4)
;
burst.onframe = function(){ p.mouseMoved() };
burst.load('ninja');
burst.play();
